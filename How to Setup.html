🎉✨ YTDM - YouTube Download Manager (Windows 10 / 11 Setup) ✨🎉

Hello! 👋 Thanks for choosing YTDM - a simple, lightweight YouTube downloader for Windows.

📝 What is YTDM?

This YouTube downloader lets you easily save audio and video content from YouTube. It supports high-quality formats and preserves metadata and thumbnails, so your files are ready for offline use or personal enjoyment. 🎧📽️

🔧 Step 1 - Install Python

  • Go to: python.org/downloads and download the latest stable Python installer for Windows. 🌐⬇️
  • Open your *Downloads* folder and double-click the downloaded installer
  • ✅ Tick “Install launcher for all users” and “Add Python to PATH”.
  • ⚠️ When you click **Install Now**, Windows may show a **UAC (User Account Control)** prompt asking for administrative permission. Click **“Yes”** to allow the installer to run with administrator privileges. (This is normal and required for some installation tasks.) 🔒👍
  • After installation, click Close. ✅

📁 Step 2 - Setup YTDM

Open the *YTDM App* folder. You should see:

  • Folders: main, Assets, bin, change-path, help 📂
  • Files: data.json, One-Time-Setup.bat, LICENSE.txt, How to Setup.HTML, YTDM shortcut 🔎

Right-click One-Time-Setup.bat → choose **Run as administrator**. Let the script finish its work. When it completes, close the script window and restart your PC (this ensures PATH/associations are applied). 🔁💻

After restart, you can run YTDM!

💡 Tip:** You can copy the **YTDM shortcut** anywhere you like (Desktop, Start menu folder, quick-access) for easy access - just copy & paste the shortcut; then double-click it to run the software. 📎➡️🖥️

🛈 Additional Info

  • Moving the app: If you want to move YTDM to a different location, move the entire **YTDM App** folder (keep internal structure intact). After moving, run **One-Time-Setup.bat** again as administrator and then run form the shortcut. 🚚📦
  • Shortcut deleted? If your **YTDM shortcut** gets deleted, you can copy the shortcut again from the main folder. That's why it's recommended to **COPY** the shortcut, not CUT or MOVE it - copying keeps the original shortcut in the app folder as a backup. 🔁🛡️
  • Help: Use the in-app Help button to open help 🆘📘
